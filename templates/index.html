Welcome to the Anime API Service!

This is a Opem Source API Service for Anime ranking and information. In this section you can find the documentation for the API. Please be sure to read it before using the API.

Documentation

Method URL Description
GET /anime Get all anime
GET /anime/{id} Get a single anime
POST /anime Create a new anime
PUT /anime/{id} Update an anime
DELETE /anime/{id} Delete an anime
